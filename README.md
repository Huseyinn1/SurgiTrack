# SurgiTrack
## Proje Tanımı
Erciyes Üniversitesi Hastanesi Çocuk Polikliniği için geliştirilen SurgiTrack projesi, ameliyat takibi ve yönetimi için ASP.NET Core ve EF Core kullanılarak oluşturulmuş ve N katmanlı mimari kullanılmış bir uygulamadır. Veritabanı yönetimi için MSSQL tercih edilmiştir. Kullanıcı arayüzü için HTML ve CSS kullanılmıştır. Projede ayrıca SonarQube ile güvenlik testleri yapılmış ve Postman ile API testleri gerçekleştirilmiştir.
